Постраничная выборка данных (paging)

Нижеследующий пример постраничной выборки данных демонстрирует следующую логику выполнения запроса (выбирается "страница" записей 4-7):

  • SELECT - Выборка выполняется из справочника "Номенклатура"
  • ORDER  - Выбранные записи сортируются по реквизиту "Код"
  • OFFSET - Первые три, из выбранных по порядку, записи отбрасываются
  • FETCH  - Выбираются следующие четыре записи после пропущенных трёх

    SELECT Ссылка, Код, Наименование
      FROM Справочник.Номенклатура
  ORDER BY Код ASC
    OFFSET 3 ROWS
FETCH NEXT 4 ROWS ONLY
