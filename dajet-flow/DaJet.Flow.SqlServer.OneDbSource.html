DaJet.Flow.SqlServer.OneDbSource

Блок-источник данных, получаемых в результате выполнения
запроса к базе данных 1С:Предприятие 8 на языке DaJet QL.

Тип СУБД: Microsoft SQL Server

Входящий тип данных: НЕТ

Исходящий тип данных: System.Data.IDataRecord

Возможна трансформация исходящего типа данных в тип данных DaJet.Flow.Payload
при помощи блока-трансформатора DaJet.Flow.RecordToPayloadTransformer.
