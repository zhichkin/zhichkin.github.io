DaJet.Flow.SqlServer.Producer

Блок-приёмник данных: выполняет инструкции DML в базе данных,
а именно INSERT, UPDATE или DELETE, на языке SQL целевой СУБД.

Тип СУБД: Microsoft SQL Server

Входящий тип данных: System.Data.IDataRecord

Исходящий тип данных: НЕТ

Настройки блока Producer:

ОпцияЗначениеПример
Target Индентификатор базы данных на сервере DaJet ms-trade-demo
Script Идентификатор скрипта (запроса) базы данных,
указанной в опции Source (путь или URL)		 flow/insert-product
Timeout Таймаут выполнения запроса СУБД в секундах 5
