DaJet.Flow.RabbitMQ.Consumer

Блок-источник данных, получаемых из очереди RabbitMQ.

Тип брокера: RabbitMQ

Входящий тип данных: НЕТ

Исходящий тип данных: DaJet.Flow.RabbitMQ.Message

Возможна трансформация исходящего типа данных в тип данных System.Data.IDataRecord
при помощи блока-трансформатора DaJet.Flow.RabbitMQ.MessageToRecordTransformer.

Настройки блока Consumer (подробнее в документации VMware):

ОпцияЗначениеПример
Pipeline Идентификатор контекста выполнения конвейера DaJet Flow b957d8c9-3403-450d-8c18-c0e7b6525e55
Source URI сервера и виртуального хоста RabbitMQ amqp://guest:guest@localhost:5672/%2F
Queue Имя очереди на сервере RabbitMQ для чтения сообщений dajet-incoming-queue
Heartbeat Период проверки в секундах "жив" ли клиент RabbitMQ.
Если нет, то выполняется автоматическое переподключение.		 60
