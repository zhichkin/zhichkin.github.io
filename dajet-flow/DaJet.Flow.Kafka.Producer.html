DaJet.Flow.Kafka.Producer

Блок-приёмник данных: выполняет запись в топик Apache Kafka.

Тип брокера: Apache Kafka

Входящий тип данных: DaJet.Flow.Payload

Исходящий тип данных: НЕТ

Настройки блока Producer (подробнее в документации Confluent):

ОпцияЗначениеПример
Topic Идентификатор топика на сервере Kafka для записи данных dajet-topic
ClientId Идентификатор клиента для логирования на сервере Kafka dajet-app-2
BootstrapServers Адрес подключения к кластеру Kafka (обязательно) 127.0.0.1:9092
Acks Способ подтверждения клиенту сохранения записи сервером Kafka all
MaxInFlight Максимально допустимое количество неподтверждённых сервером Kafka
сообщений (см. Acks). Важно для гарантии последовательности FIFO.		 1
MessageTimeoutMs Таймаут ожидания клиентом отчёта о доставке сообщения от сервера Kafka. 30000
EnableIdempotence Включение механизма дедупликации сообщений на сервере Kafka. false
